# EHD产品使用方法与护理指南

## 一、EHD防脱固发洗发水 使用方法

### 基础使用步骤

1. **充分湿发**：用温水（约38-40°C）将头发和头皮彻底打湿，水温不宜过热，过热会刺激头皮油脂分泌
2. **取适量产品**：短发取约1元硬币大小，中长发取约2元硬币大小，长发可适量增加
3. **掌心起泡**：将洗发水在掌心搓出泡沫后再涂抹到头皮上，避免直接将洗发水倒在头皮上
4. **指腹按摩**：用指腹（非指甲）轻柔按摩头皮2-3分钟，重点按摩头顶、两侧太阳穴、后脑勺等容易脱发的区域
5. **停留片刻**：让泡沫在头皮上停留1-2分钟，使活性成分充分作用
6. **彻底冲洗**：用温水彻底冲洗干净，确保无残留

### 进阶使用技巧（推荐）

- **二次清洁法**：第一遍快速清洗去除表面油脂和灰尘（约30秒），第二遍重点按摩头皮让成分吸收（2-3分钟）。适合头皮较油或2天以上未洗头的情况
- **建议使用频率**：每2-3天使用一次，油性头皮可隔天使用。不建议每天使用，给头皮适当的自我调节时间
- **最佳使用时间**：晚间洗头优于早晨，洗后头皮毛孔打开有利于夜间修复

### 见效周期

- 第1-2周：头皮出油情况改善，洗后清爽感增强
- 第3-4周：掉发量逐渐减少（洗头时掉发明显减少）
- 第6-8周：新生细软绒毛开始出现（因人而异）
- 持续使用3个月以上效果更佳

**注意**：个体差异较大，以上周期仅供参考。如使用8周以上无明显改善，建议就医排查病理性脱发。

## 二、EHD控油祛痘洗发水 使用方法

### 使用步骤

1. 温水充分湿发
2. 取适量产品在掌心起泡
3. 涂抹至头皮，指腹按摩1-2分钟，重点清洁出油严重和有痘痘的区域
4. 彻底冲洗干净

### 使用频率

- 重度油头：可每天使用
- 中度油头：隔天使用
- 轻度油头：每2-3天使用一次
- 头皮有痘痘期间：建议每天或隔天使用，直到症状改善后降低频率

## 三、EHD滋养修护发膜 使用方法

### 使用步骤

1. 先用洗发水清洗头发并冲洗干净
2. 将头发多余水分轻轻拧干（半干状态吸收最佳）
3. 取适量发膜（核桃大小），均匀涂抹在发丝中段至发尾，**避开头皮和发根**
4. 用手指或宽齿梳将发膜梳理均匀
5. 停留5-10分钟（可用热毛巾包裹加强吸收效果）
6. 用温水彻底冲洗干净

### 使用频率

- 正常发质：每周1次
- 干枯受损发质：每周2次
- 烫染后修护期：前两周可每周2-3次，之后恢复每周1-2次

**注意**：发膜不能替代护发素日常使用，也不建议每天使用，过度使用可能导致头发过于软塌。

## 四、EHD护发精油 使用方法

### 使用场景与方法

**场景一：洗后护理（湿发使用）**
1. 洗完头发后用毛巾轻轻吸干多余水分
2. 取2-3滴精油于掌心搓热
3. 从发丝中段向发尾方向涂抹，避开头皮
4. 可自然晾干或吹风机吹干

**场景二：日常打理（干发使用）**
1. 取1-2滴精油于掌心搓热
2. 轻轻抹在毛躁的发尾或飞翘的碎发上
3. 用于出门前整理造型，抚平毛躁

**场景三：吹风前保护**
1. 洗后半干状态取2-3滴涂抹发尾
2. 再用吹风机吹干，精油形成保护膜减少热损伤

### 用量参考

| 发长 | 建议用量 |
|------|---------|
| 短发（耳上） | 1-2滴 |
| 中发（肩部） | 2-3滴 |
| 长发（肩下） | 3-5滴 |

## 五、产品搭配使用建议

### 推荐搭配方案

**方案一：防脱控油全套护理（油性脱发人群）**
- 日常：防脱固发洗发水（隔天）
- 交替：控油祛痘洗发水（每周1-2次深层清洁）
- 每周：滋养修护发膜（1次，仅涂发尾）
- 日常：护发精油（洗后涂发尾）

**方案二：干枯脱发修护方案（干性脱发人群）**
- 日常：防脱固发洗发水（每2-3天）
- 每周：滋养修护发膜（2次）
- 日常：护发精油（每次洗后+日常打理）

**方案三：烫染后修护方案**
- 日常：防脱固发洗发水（每2-3天）
- 每周：滋养修护发膜（前两周每周3次，之后每周1-2次）
- 日常：护发精油（每次洗后必用，保护受损发丝）

### 使用顺序

正确顺序：洗发水 → 冲洗 → 发膜 → 停留5-10分钟 → 冲洗 → 吹至半干 → 护发精油
