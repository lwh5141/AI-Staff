# EHD适用人群与使用场景

## 一、公开页能支持的目标人群

### 高可信目标人群
- 压力掉发人群
- 发质脆弱、易断发人群
- 发量稀少困扰人群
- 细软塌、追求蓬松感人群
- 头皮易出油人群
- 男女通用日常洗护人群

### 低至中可信、仅可标注为公开宣称场景
- 发际线后移
- 发缝变宽
- 产后脱发
- 熬夜掉发
- 烫染后脆弱发丝
- 换季头皮状态不稳

## 二、推荐的知识库标准场景标签

```yaml
scenarios:
  - 压力掉发
  - 细软扁塌
  - 发丝脆弱易断
  - 头皮出油
  - 追求蓬松感
  - 日常防脱养护
```

## 三、分场景应答模板

### 场景1：用户说“我头发细软塌，还容易油”
推荐回答：
EHD 公开卖点里对“控油蓬松、强根健发、细软塌发质”覆盖较多，属于比较匹配的使用场景。若你更关心的是发根支撑感和洗后蓬松感，可以优先关注它的控油蓬松版/500ml版描述。

### 场景2：用户说“我主要是掉发严重”
推荐回答：
EHD 的主打方向是“防脱固发 + 减少断发”，公开渠道里也反复出现“4周有效防脱”等卖点。但这类表述更接近商品宣称，知识库回复时最好同步提示：如果是突然大量脱发、斑秃或持续加重，建议尽快就医排查病理因素。

### 场景3：用户说“我只想找日常洗头不那么塌的”
推荐回答：
从公开页面看，EHD 不只是强调防脱，也强调控油蓬松与日常洗护，因此对于“头发容易贴头皮、想洗后更蓬松”的用户也属于可考虑场景。

## 四、不建议模型直接做医学判断
以下情况出现时，知识库应该转人工/建议就医，而不是继续推荐商品：
- 短期内突然大量脱发
- 斑秃、局部明显秃斑
- 头皮红肿、渗液、剧烈瘙痒
- 正在孕期/哺乳期且需确认配方安全
- 与药物治疗并用问题
