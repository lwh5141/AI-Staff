# EHD控油祛痘洗发水 产品详情

## 基本信息

- 产品全称：EHD控油祛痘清爽洗发水（Oil Control Acne Clearing Shampoo）
- 产品编码（SKU）：EHD010001
- 规格：500ml
- 保质期：未开封3年，开封后建议12个月内用完
- 产地：中国广东
- 适用人群：头皮油腻、头皮长痘/毛囊炎、油脂性头屑、头皮瘙痒人群
- 适用发质：油性发质
- 质地：透明凝胶状
- 香型：清凉薄荷草本香

## 核心卖点

1. **深层控油**：柳树皮提取物（天然水杨酸）深层清洁毛孔，溶解多余皮脂和老旧角质
2. **祛痘抑菌**：OCT成分（吡罗克酮乙醇胺盐）有效抑制马拉色菌，改善头皮痘痘和毛囊炎
3. **平衡油水**：多种植物提取物协同调节头皮油脂分泌，不是简单的"去油"而是"控油平衡"
4. **温和清洁**：不含皂基，清洁力强但不过度去脂，避免越洗越油的恶性循环
5. **清爽蓬松**：洗后头发蓬松不塌，控油效果可维持24-48小时

## 产品功效说明

- 深层清洁头皮多余油脂和角质堆积
- 抑制头皮痘痘相关细菌，改善头皮毛囊炎
- 去屑止痒，改善油脂性头屑
- 调节头皮水油平衡，延长清爽时间
- 改善因油脂堵塞毛囊导致的掉发问题

**重要声明**：本产品为化妆品，非药品。严重毛囊炎或头皮皮肤病请就医治疗。

## 与防脱固发洗发水的区别

| 对比项 | 防脱固发洗发水 | 控油祛痘洗发水 |
|--------|--------------|--------------|
| 核心功效 | 防脱固发、强韧发根 | 控油祛痘、去屑止痒 |
| 主打成分 | 侧柏叶+当归+锯棕榈果 | 柳树皮+OCT+苦参 |
| 适用人群 | 脱发/掉发严重人群 | 头皮油腻/头皮痘痘人群 |
| 控油力度 | 中等 | 强 |
| 防脱力度 | 强 | 中等（通过控油间接改善） |
| 适合发质 | 各类发质 | 油性发质为主 |
| 使用频率 | 每2-3天一次 | 可每天使用（油性头皮） |

## 使用建议

- 油性头皮且伴有脱发：建议防脱固发洗发水与控油祛痘洗发水交替使用
- 单纯油性头皮无明显脱发：推荐使用控油祛痘洗发水
- 头皮有痘痘/毛囊炎：优先使用控油祛痘洗发水，症状严重请就医
