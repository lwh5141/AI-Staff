# EHD店铺售后政策

## 一、退货退款政策

### 7天无理由退货

- **适用条件**：自签收之日起7天内，商品未开封、未使用、包装完好、不影响二次销售
- **退货流程**：
  1. 在订单详情页点击"申请退货"
  2. 选择退货原因，提交申请
  3. 客服审核通过后，会提供退货地址
  4. 买家自行寄回商品（建议使用有物流跟踪的快递）
  5. 仓库收到退货并验收通过后，3-5个工作日内退款至原支付账户
- **退货运费**：
  - 因商品质量问题退货：运费由店铺承担
  - 因个人原因退货（不想要/买错等）：运费由买家承担

### 已开封商品退货规则

- **原则上已开封商品不支持7天无理由退货**
- **例外情况**：
  - 产品存在质量问题（如漏液、变质、异味异常等）：开封后仍可退货，需提供照片/视频证据
  - 过敏反应：需提供过敏部位照片及医院诊断证明，经审核后可办理退货

### 超过7天的退货处理

- 签收超过7天但不超过15天：
  - 有质量问题：可申请退换货，需提供相关证据
  - 无质量问题：原则上不予退货
- 签收超过15天：不接受退货申请（质量问题除外，需与客服协商）

## 二、换货政策

- 收到商品后7天内发现质量问题可申请换货
- 换货流程与退货类似，客服审核通过后安排换货
- 质量问题换货运费由店铺承担
- 换货商品为同款同规格，如需更换其他产品请走退货重新下单流程

## 三、质量问题处理

### 常见质量问题及处理方式

| 问题类型 | 处理方式 | 所需材料 |
|---------|---------|---------|
| 产品漏液/破损 | 全额退款或补发 | 外包装及漏液照片 |
| 产品变质（异味/变色） | 全额退款或换货 | 产品照片+批号照片 |
| 收到空瓶/少件 | 补发缺少商品 | 开箱视频/快递面单照片 |
| 产品与描述不符 | 协商退款或换货 | 产品实物照片 |
| 使用后过敏 | 协商处理 | 过敏照片+医院证明 |

### 重要提醒

- **开箱建议**：收到快递后建议录制开箱视频，如遇到商品破损/少件等问题可作为有效凭证
- **保留包装**：退换货需要保留原包装，无原包装可能影响退换货处理
- **及时反馈**：发现问题请在签收后48小时内联系客服，超时可能影响处理效率

## 四、发票开具

- 支持开具电子普通发票和电子增值税专用发票
- 下单时可在备注中注明开票需求，或联系客服提供开票信息
- 发票内容：日化用品/化妆品
- 发票将在发货后5个工作日内发送至指定邮箱
- 如需纸质发票，可联系客服单独申请（额外需3-5个工作日邮寄）

## 五、客服联系方式

- **在线客服**：店铺首页点击"联系客服"，工作时间9:00-22:00
- **售后热线**：400-XXX-XXXX（工作日9:00-18:00）
- **响应时效**：在线咨询即时响应，售后问题24小时内给出处理方案
