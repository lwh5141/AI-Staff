# EHD 防脱洗发水 RAG 知识包（公开网页整理版）

生成日期：2026-04-08

## 交付说明

本知识包基于**公开可见网页**整理，目标是为 RAG 项目提供可直接入库的结构化文档，而不是替代品牌官方法务/品控确认稿。

### 本包的适用场景
- 电商客服知识库
- 商品问答机器人
- 售前/售后 FAQ
- 运营检索问答
- 商品详情摘要/多平台信息对齐

### 证据分级
- **A级：平台/监管/商家公开页**  
  如：国家药监局、Yami、Weee、MaskQueen、苏宁公开页、抖音官方店铺主页
- **B级：公开聚合页/媒体转载页**  
  如：淘宝/天猫优惠聚合页、都市快报公开稿等
- **C级：公开讨论/种草/二次传播页**  
  仅用于补充“常见说法”，默认不作为强事实

### 使用限制
1. **淘宝/天猫/抖音原始商品详情页存在反爬/登录限制**，因此部分内容来自可公开访问的检索快照、聚合页、店铺主页或二级销售页。  
2. **成分表当前只能确认到“部分公开成分/公开卖点成分”**，未获得完整包装背标全成分图，因此本包已将“已确认成分”与“公开宣称成分”分开。  
3. **监管信息存在冲突点**：`国妆特字20241933` 历史批准页可检索到，但当前国家药监局二维码页显示“已作废”。入生产知识库前，建议人工复核最新注册状态。  
4. 不同平台出现 **400ml / 500ml / 300ml / 100ml** 等不同规格，问答时应优先按 SKU 区分。

### 推荐入库策略
- 直接按 Markdown 入库
- 分块粒度建议：每个 `##` 标题下单独分块
- 元数据建议保留：
  - `source_scope`
  - `trust_level`
  - `last_checked`
  - `sku_variant`
  - `doc_type`
  - `status`
